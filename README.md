# 💰 Mini Sistema Bancário em Java (Backend)

Status do Projeto: 🚀 Concluído

## 📝 Sobre o Projeto
Este projeto simula as operações básicas de uma conta bancária através do terminal. O sistema permite ao usuário realizar depósitos, saques, consultar o saldo atual e visualizar o fluxo das transações, aplicando regras de negócio simples (como impedir saques maiores do que o saldo disponível).

## 🛠️ Tecnologias e Conceitos Utilizados
- **Linguagem:** Java
- **Regras de Negócio e Validações:** Lógica condicional avançada para controle de saldo e validação de saques e depósitos.
- **Estruturas de Controle:** Laço de repetição (`while`) para o menu interativo e `switch-case` para direcionar as ações do cliente (sacar, depositar, ver saldo, sair).
- **Interação com o Usuário:** Classe `Scanner` para entrada de dados via console.
